## 如何使用 config.json：

在您的項目文件夾中創建一個名為“config.json”的文件。 它將具有以下語法：

```
{
  "token": "bot token here",
  "prefix": "default command prefix here",
  "mongoPath": "mongo path here",
  "redisPath": "redis path here"
}
```
